# 회의 출석 안내서 (실외 모드)

> GPS 위치 확인으로 출석 체크하는 방법을 안내합니다.
>
> 어려운 점이 있으시면 회의 담당자에게 편하게 문의해주세요.

---

## 준비물

- **스마트폰** (인터넷 연결 필요)
- **위치 서비스(GPS)** 켜기

---

## 출석 체크 방법

### 1단계. 출석 페이지 열기

담당자가 보여주는 **QR 코드**를 스마트폰 카메라로 비춰주세요.

자동으로 출석 페이지가 열립니다.

![QR 코드를 카메라로 스캔하는 화면](images/outdoor_01_scan_qr.png)

> QR 코드가 잘 안 되시면, 담당자가 알려주는 **인터넷 주소(링크)** 를 직접 눌러도 됩니다.

---

### 2단계. 위치 권한 허용하기

페이지가 열리면 아래와 같은 팝업이 나타납니다.

![위치 권한 허용 팝업 화면](images/outdoor_02_location_permission.png)

**반드시 "허용"을 눌러주세요.**

- "허용"을 누르지 않으면 출석 체크가 되지 않습니다.
- 실수로 "차단"을 누르셨다면 → [문제 해결 방법](#위치-권한을-차단했어요) 참고

---

### 3단계. 위치 확인 기다리기

화면에 **"위치를 확인하고 있습니다..."** 라는 메시지가 나타납니다.

잠시만 기다려주세요. (보통 5~10초)

![위치 확인 중 화면](images/outdoor_03_checking_location.png)

초록색으로 **"위치 확인 완료"** 가 나타나면 성공입니다!

![위치 확인 완료 화면](images/outdoor_04_location_confirmed.png)

> 빨간색으로 **"회의실에서 OOm 떨어져 있습니다"** 라고 나오면
> → [위치가 멀다고 나와요](#위치가-멀다고-나와요) 참고

---

### 4단계. 내 정보 입력하기

아래 3가지를 입력합니다.

![정보 입력 화면](images/outdoor_05_input_form.png)

| 항목 | 입력 예시 | 꼭 입력해야 하나요? |
|------|----------|-------------------|
| **성명** | 홍길동 | 네, 필수입니다 |
| **소속** | 경영학과 | 네, 필수입니다 |
| **이메일** | hong@yiu.ac.kr | 아니요, 선택입니다 |

- **성명**: 본인의 이름을 정확히 입력해주세요.
- **소속**: 학과 또는 부서 이름을 입력해주세요.
- **이메일**: 입력하시면 출석 확인 메일을 받으실 수 있습니다.

---

### 5단계. "출석 체크하기" 버튼 누르기

정보를 모두 입력했으면, 아래쪽의 큰 버튼을 누릅니다.

![출석 체크하기 버튼](images/outdoor_06_submit_button.png)

**"출석 정보를 전송하고 있습니다..."** 메시지가 나타나면 잠시 기다려주세요.

---

### 6단계. 출석 완료!

아래와 같은 **출석 확인증**이 나타나면 출석이 완료된 것입니다.

![출석 확인증 화면](images/outdoor_07_certificate.png)

확인증에는 다음 정보가 표시됩니다:

- **확인 번호** (예: 260210-1234)
- **성명**
- **소속**
- **출석 시간**
- **회의명**

---

## 출석 확인증 저장하기

출석 확인증은 나중에 증빙 자료로 사용할 수 있습니다.

**꼭 저장해 두세요!**

1. 확인증 아래의 **"확인증 저장"** 버튼을 누릅니다.
2. 이미지 파일이 스마트폰에 저장됩니다.
3. **갤러리(사진첩)** 에서 확인하실 수 있습니다.

![확인증 저장 버튼](images/outdoor_08_save_button.png)

> 저장이 안 되시면, 화면을 **캡처(스크린샷)** 하셔도 됩니다.
>
> - **아이폰**: 전원 버튼 + 볼륨 올리기 버튼 동시에 누르기
> - **갤럭시**: 전원 버튼 + 볼륨 내리기 버튼 동시에 누르기

---

## 내 출석 조회하기

출석이 잘 되었는지 다시 확인하고 싶을 때 사용합니다.

1. 출석 페이지 **맨 아래로** 스크롤합니다.
2. **"내 출석 확인"** 부분을 누릅니다.
3. **성명**과 **소속**을 입력합니다.
4. **"출석 조회"** 버튼을 누릅니다.

![출석 조회 화면](images/outdoor_09_check_attendance.png)

- **초록색** 메시지가 나오면 → 출석이 정상 처리된 것입니다.
- **노란색** 메시지가 나오면 → 이름과 소속이 정확한지 다시 확인해주세요.

---

## 화면 모드 바꾸기

화면이 너무 밝거나 어두우시면 **오른쪽 위**의 버튼을 눌러 바꿀 수 있습니다.

- 해 모양 → 밝은 화면
- 달 모양 → 어두운 화면

![다크모드 전환 버튼](images/outdoor_10_darkmode.png)

## 문제 해결 방법

### 위치 권한을 차단했어요

스마트폰 설정에서 다시 허용할 수 있습니다.

**아이폰:**
1. 설정 앱 열기
2. Safari (또는 사용 중인 브라우저) 선택
3. 위치 → "허용"으로 변경

**갤럭시:**
1. 설정 앱 열기
2. 앱 → Chrome (또는 사용 중인 브라우저) 선택
3. 권한 → 위치 → "허용"으로 변경

설정 변경 후 출석 페이지를 **새로고침** 해주세요.

---

### 위치가 멀다고 나와요

GPS 오차로 실제 위치와 다르게 표시될 수 있습니다.

![위치 초과 시 지도 화면](images/outdoor_11_location_map.png)

**이렇게 해보세요:**

1. **창가 쪽으로** 이동해주세요 (GPS 신호가 더 잘 잡힙니다).
2. 스마트폰의 **Wi-Fi를 켜 주세요** (위치 정확도가 올라갑니다).
3. 화면의 **"위치 다시 확인하기"** 버튼을 눌러주세요.

여러 번 시도해도 안 되시면 **담당자에게 말씀해주세요**.

---

### 이미 출석했다고 나와요

한 스마트폰에서는 **한 번만** 출석 체크가 가능합니다.

이미 출석하셨다면 확인증이 자동으로 표시됩니다.

---

### 화면이 안 열려요

1. **인터넷 연결**을 확인해주세요.
2. 다른 브라우저(크롬, 사파리)로 시도해주세요.
3. 그래도 안 되면 담당자에게 문의해주세요.

---

### 그래도 해결이 안 되면

**회의 담당자에게 편하게 말씀해주세요.**

담당자가 직접 출석 처리를 도와드릴 수 있습니다.
